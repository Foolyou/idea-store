## 1. 设计理念

| 关键词 | 含义 | 设计投射 |
|--------|------|----------|
| 轻盈 | 每一步操作零摩擦，记录灵感不应有门槛 | 最少输入、最大留白、单手可触达 |
| 温润 | 平台是灵感的温床，非冰冷的工具 | 暖色调、圆角、柔和阴影 |
| 克制 | 不做社交喧嚣，让内容说话 | 弱化界面 chrome，动画仅用于反馈 |

风格基调：**温暖、柔软、留白**。

大地调 — 陶土与木质的自然质感，色温偏暖、灰度适中。

## 3. 字体系统

### 3.1 字体族

```css
font-family: "PingFang SC", "Microsoft YaHei", system-ui, -apple-system, sans-serif;
```

优先使用系统原生中文字体，保证渲染质量与零额外加载。

### 3.2 字阶

| 层级 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| 标题 | 20px | SemiBold 600 | 1.4 | 页面标题、卡片标题 |
| 正文 | 16px | Regular 400 | 1.75 | 灵感正文 |
| 辅助 | 13px | Regular 400 | 1.5 | 时间戳、昵称、标签、统计数据 |

## 4. 间距系统

基于 4px 网格，确保垂直韵律一致。

| Token | 值 | 用途 |
|-------|----|------|
| `--space-xs` | 4px | 紧凑元素间距 |
| `--space-sm` | 8px | 组件内部元素间距 |
| `--space-md` | 12px | 卡片间距 |
| `--space-lg` | 16px | 卡片内边距、页面水平内边距 |
| `--space-xl` | 20px | 段落间距 |
| `--space-2xl` | 24px | 大区块间距 |
| `--space-3xl` | 32px | 页面顶部/底部留白 |

## 5. 形状系统

| Token | 值 | 用途 |
|-------|----|------|
| `--radius-sm` | 8px | 标签、小按钮、输入框内圆角 |
| `--radius-md` | 14px | 卡片、输入框、面板 |
| `--radius-lg` | 20px | 弹窗、大面板、底部 Sheet |
| `--radius-full` | 9999px | 头像、胶囊按钮、菜单球 |

### 5.1 阴影

| Token | 值 | 用途 |
|-------|----|------|
| `--shadow-card` | `0 2px 12px rgba(120,90,60,0.06)` | 卡片投影 |
| `--shadow-float` | `0 4px 16px rgba(120,90,60,0.12)` | 浮动元素（菜单球、弹窗） |

阴影统一使用大地色系（#785A3C），保持温暖质感，避免纯黑投影的生硬感。

## 7. 交互状态

### 7.1 发布灵感

```
[输入态]              占位符「此刻的想法…」
   │
   ├─ 添加图片 ──→ [图片已选]    缩略图展示，点击可移除，最多9张
   │
   ├─ 切换范围 ──→ [范围面板]    Bottom Sheet：
   │                   公开 / 圈子（选择圈子）/ 私人
   │                   默认继承上次配置
   │
   └─ 点击发送 ──→ [发布中]      按钮 loading，禁用重复点击
         │
         ├─ 私人/圈子 ──→ [完成]  内容出现在对应流顶部
         │
         └─ 公开 ──→ [完成+Toast] 「已公开发布」[关闭分享]
                       点击关闭 → 灵感变为私人 → Toast消失
```

### 7.2 浮动菜单球

```
[收起态]                    首页右下角 fixed，48px 圆形
   │
   └─ 点击 ──→ [展开态]      蒙层出现（rgba 0,0,0,0.4）
                │             菜单球变形为关闭按钮（✕）
                │             菜单面板右下角弹出
                │
                ├── 我的灵感 ──→ /me
                ├── 我的圈子 ──→ /me
                ├── 收藏夹   ──→ /me
                ├── 草稿箱   ──→ /me
                ├── 发现圈子 ──→ /circles
                └── 账号设置 ──→ /me
                     │
[蒙层点击 / 点击关闭] ──→ [收起态]
```

### 7.3 圈子加入

```
[社区流] 灵感上的圈子标签 ──→ 点击 ──→ 圈子页
[菜单球] → 发现圈子 ──→ 圈子列表 → 点击圈子 ──→ 圈子页
                                                  │
                                        [未加入] 底部「加入圈子」按钮
                                        [已加入] 顶部发布组件（锁定本圈）
                                                  │
                                        ──「加入」──→ 即时成为成员
```

### 7.4 社区流浏览

- 首页首屏加载 20 条（公开 + 已加入圈子的内容），按时间倒序
- 滚动至距底部 200px 时自动加载下一页 20 条
- 无更多内容时底部显示「已经到底了」
- 支持下拉刷新

### 7.5 灵感详情

- 社区流点击灵感卡片（非按钮区域）→ 推入详情页
- 详情页展示完整内容 + 发布者 + 点赞/收藏 + 圈子标签
- 左上角返回或系统手势返回，回到之前的滚动位置

### 7.6 草稿恢复

- 输入内容实时自动保存
- 再次打开首页时自动回填上次内容、图片、发布范围
- 手动清空输入框后草稿覆盖为空白

### 7.7 灵感编辑与删除

- 入口：个人管理页左滑卡片，或详情页右上角更多按钮
- 编辑：发布区填充当前内容 → 修改 → 「保存修改」→ 原地更新
- 删除：确认弹窗「确定删除这条灵感？」→ 确认后列表移除

### 7.8 空状态

| 页面/区域 | 空状态文案 | 引导操作 |
|-----------|-----------|----------|
| 首页社区流 | 还没有灵感，去发现一些有趣的圈子吧 | [去发现圈子] |
| 我的灵感 | 还没有发布灵感 | [去发布] |
| 收藏夹 | 还没有收藏灵感 | [去发现] |
| 草稿箱 | 暂无草稿 | — |
| 圈子列表 | 还没有圈子，创建一个吧 | [创建圈子] |
| 圈子成员 | — | 至少显示创建者 |

## 8. 动效规范

| 场景 | 动画 | 时长 | 缓动 |
|------|------|------|------|
| 页面推入 | 从右滑入 | 250ms | ease-out |
| 页面返回 | 向右滑出 + 前页渐显 | 250ms | ease-in |
| Bottom Sheet 弹出 | 从底部滑入 | 200ms | ease-out |
| Bottom Sheet 收起 | 向底部滑出 | 150ms | ease-in |
| 菜单球展开 | 面板弹出 + 蒙层渐显 | 200ms | ease-out |
| 菜单球收起 | 面板收起 + 蒙层渐隐 | 150ms | ease-in |
| Toast 出现 | 从底部滑入 | 200ms | ease-out |
| 按钮按压 | scale 0.97 | 100ms | ease-in-out |
| 列表加载 | 骨架屏闪烁 | — | — |
| 下拉刷新 | 旋转指示器 | — | — |

| 属性 | 移动端 (375-428px) | 桌面端 (≥768px) |

- 顶部状态栏：发布区不做 notch 侵入，保持安全间距
